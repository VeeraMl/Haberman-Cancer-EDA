# Haberman Cancer Survival EDA

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Haberman Cancer Survival Dataset using Python.

The objective of this project is to analyze patient survival patterns and understand important features using visualization and statistical analysis techniques.

---

## Dataset Information

The dataset contains information about patients who underwent surgery for breast cancer.

### Features

* **Age** → Age of patient
* **Year** → Year of operation
* **Nodes** → Number of positive axillary nodes detected
* **Status**

  * 1 = Survived 5 years or longer
  * 2 = Died within 5 years

## Key Observations

* Nodes feature provides better separation between survival classes.
* Patients with fewer positive nodes had higher survival chances.
* Age and operation year show significant overlap between classes.
* The dataset is not perfectly linearly separable.

## Conclusion

Exploratory Data Analysis helped in understanding important patterns and survival distributions present in the dataset. Visualization techniques made it easier to identify useful insights from the dataset.
